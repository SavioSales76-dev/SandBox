# Contexto do Projeto

## Objetivo
Criar componentes reutilizaveis para projetos futuros, com foco em agilizar o desenvolvimento e aumentar a consistencia entre aplicacoes.

## Direcao de Trabalho
- Construir uma base de componentes Vue organizada e escalavel.
- Priorizar reutilizacao, manutencao e facilidade de integracao.
- Padronizar estrutura e boas praticas para acelerar novos projetos.

## Experiencia da Tela de Componentes
- Cada celula de componente na tela de `Buttons` deve expor um botao `Get Code`.
- O `Get Code` deve levar para uma view dedicada daquele componente.
- A view dedicada deve exibir o codigo do componente para consulta e copia.
- O contador de `Views` em cada celula deve representar a quantidade de entradas na view dedicada (nao cliques no componente visual).
- Esse padrao deve servir como template para todos os componentes futuros da biblioteca.

## Referencia
Este arquivo deve servir como contexto principal do que estamos desenvolvendo neste repositorio.
